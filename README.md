# Ақсерек · Көксерек

Интерактивная 3D-игра по мотивам казахской народной игры «Ақсерек-көксерек». Проект переносит традиционные правила в формат зрелищной сцены с командным выбором, тайминг-миниигрой, AI-соперником, голосовой озвучкой и поддержкой мобильных устройств.

**Live:** https://akserek-kokserek.vercel.app

---

## О игре

Игрок управляет командой «Ақсерек» и сражается против «Көксерек». В каждом раунде выбирается бегун и цель в цепи соперника. Исход прорыва определяется силой персонажей и точностью попадания в зелёную зону тайминг-бара. После каждого раунда — комментарий AI-аташки.

### Как играть

1. Выбери сложность и тип соперника (бот / AI)
2. Выбери профиль команды — атака, баланс или защита
3. **ШАГ 1:** выбери своего бегуна из левой панели
4. **ШАГ 2:** выбери цель в цепи врага из правой панели
5. Нажми **ПРОБЕЛ** в зелёной зоне тайминг-бара
6. Победа — когда у соперника остаётся ≤ 1 игрока

## Ключевые возможности

### Геймплей
- Две команды по 5 игроков, цепи, прорыв и захват
- Тайминг-миниигра для атаки и защиты
- Три профиля команды: ⚔️ атака / 🏹 баланс / 🛡 защита
- Четыре уровня сложности: `easy` / `normal` / `hard` / `impossible`
- Адаптивная сложность — игра следит за win-rate и подстраивается

### AI и контент
- **AI-соперник** (OpenAI) — стратегически выбирает бегуна и цель
- **AI-чат** — переписка с соперником прямо в игре
- **Голосовой комментатор** (ElevenLabs) — аташка озвучивает каждый раунд
- **Голосовые команды** — управление игрой через микрофон
- Фоллбэк-контент (имена, кличи, комментарии) без AI

### 3D-сцена
- Казахская степь: юрты, горы, кострище, повозка, лошади, орёл
- Алтыбакан, зрители, фестивальный декор
- Атмосферные частицы, туман, постпроцессинг (Bloom, Vignette)
- Динамическая погода: солнце, дождь, туман, ночь, шторм
- Камера меняет позицию в зависимости от фазы игры

### UX
- Полная поддержка мобильных устройств (отдельный HUD и тайминг-бар)
- Интерфейс на казахском и русском языках
- Тёмная тема, казахские орнаменты, шрифт Cinzel Decorative
- Настройки громкости прямо в игре

## Переменные окружения

Создай файл `.env` (для серверных API-роутов):

```env
# OpenAI — ходы AI-соперника, чат, комментарии
OPENAI_API_KEY=sk-...
OPENAI_MODEL=gpt-4o-mini

# ElevenLabs — голосовая озвучка
ELEVENLABS_API_KEY=...
ELEVENLABS_VOICE_ID=...
```

> Если ключи не указаны — игра автоматически переключается на фоллбэк-логику (бот без AI, текст без озвучки).
